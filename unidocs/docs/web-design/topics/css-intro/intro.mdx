# Въведение в CSS

---

## Предимства и развитие

**CSS (Cascading Style Sheets)** служи за добавяне на стилове към HTML страниците. Използването на външни стилове спестява много работа, защото те могат да се прилагат към няколко страници. Не е било предвидено HTML да съдържа тагове за форматиране на уеб страниците.
Използван е единствено, за да опише съдържанието на уеб страница.

Когато тагове като `<font>` и цветни атрибути са били добавени към HTML 3.2,  разработването на големи уеб сайтове се превърнало в дълъг и скъп процес. За да реши този проблем, световният уеб консорциум (W3C) е създал CSS.

---

## История и развитие

- **CSS1 (1996) -** Основни възможности – цветове, шрифтове, подравняване.
- **CSS2 (1998) -** Добавя позицииране, медийни типове, поддръжка за принтиране.
- **CSS2.1 (2004) -** Поправки и стабилизация на CSS2.
- **CSS3 (2011 – ...) -** Разделен на модули – анимации, преходи, flexbox, grid и др.
- **CSS4 (в развитие) -** Няма официална версия CSS4, но терминът се използва неофициално за нови CSS модули.

## Синтаксис

```csharp
h1 {
    color: blue;
}
```

- `h1` - селектор.
- `color: blue` - атрибут и неговата стойност.

Селекторите, които ще използваме на този етап са:
- Елемент;
- Идентификатор (id);
- Клас (class);

## Стилове

Можем да прилагаме стилове по различни начини:

- Inline (вграден);
- Internal (вътрешен);
- External (външна връзка);

### Inline

Стилът се вгражда в отварящия таг на конкретния елемент.

```csharp
<p style="color:red;">Параграф</p>
```

Този начин за добавяне на стил е с най-голям приоритет измежду трите.

### Internal

При този стил добавяме стиловете в тага `<head>`, а след това в тага `<style>`.

```csharp
<head>
    <style>
        h1 {
            color: red;
        }
        p {
            background-color: blue;
            text-align: center;
        }
    </style>
</head>
```

### External

Стиловете са написани в отделен външен файл, като към него се добавя връзка. Няколко страници могат да използват един и същ външен стил, което е полезно, когато искаме да преизползваме стилове за повече от 1 страница. Външните стилове са с най-малък приоритет.

**Връзка**
```csharp
<head>
    <link href="styles.css" rel="stylesheet" />
</head> 
```

**styles.css**
```csharp
p {
    color: blue;
    font-size: 30px;
    text-align: center;
}

img {
    height: 100px;
    width: auto;
}

...
```
