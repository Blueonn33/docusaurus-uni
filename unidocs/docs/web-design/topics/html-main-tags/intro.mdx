# Основни тагове в HTML

В тази страница ще бъдат разгледани най-често използваните тагове в разработката на уеб сайтове.

---

## Текстови тагове

Работата с текст означава използване на тагове за заглавия, параграфи, както и форматиране на текста. Основните тагове са:

```HTML
<!-- Заглавие -->
<h1> до <h6>

<!-- Параграф -->
<p>

<!-- Удебеляване на текста -->
<b> / <strong>

<!-- Курсив -->
<i> / <em>

<!-- Нов ред -->
<br>

<!-- Хоризонтална линия -->
<hr>

<!-- Текст (inline елемент) -->
<span>

<!-- Цветово маркиране на текста -->
<mark>
```

## Цветове

Добавянето на стойност за цвят може да стане по няколко начина:

- **Чрез име на цвета** - red, green, blue.
- **Чрез използване на RGB** - rgb(255, 0, 0), rgb(120, 120, 120). 
- **Чрез използване на HEX** - #FF0000, #431264.
- **Чрез използване на HSL** - hsl(17, 100%, 50%).

RBG и HSL си имат разширения, които използват алфа стойност, определяща прозрачността на цвета:

- **RBGA** - rgba(244, 200, 100, 0.5).
- **HSLA** - hsla(33, 50%, 50%, 1).

## Линкове

Линковете ни позволяват да се придвижваме от страница на страница. Не е задължително линкът да е текст. Той може да е снимка или друг HTML елемент. За създаване на линкове използваме тага `<a>`.

```HTML
<a href="https://www.w3schools.com/" target="_blank">Цъкни</a>
```

### Атрибутът target

По подразбиране, като отворим някой линк, новата страница се отваря в текущата. 
`Target` може да има следните стойности:

- **_self** - Отваря страницата в същия прозорец.
- **_blank** - Отваря страницата в нов прозорец.
- **_parent** - Отваря страницата в родителската рамка.
- **_top** - Отваря страницата в пълен размер на екрана.

### Абсолютни и относителни пътища

- Абсолютните пътища използват целия уеб адрес ("https://www...").
- Относителните не използват частта с "https", тъй като те водят до страници в текущия сайт.

#### Пример

```HTML
<h2>Абсолютни пътища</h2>
<p><a href="https://www.w3.org/">W3C</a></p>
<p><a href="https://www.google.com/">Google</a></p>

<h2>Относителни пътища</h2>
<p><a href="html_images.asp">HTML Снимки</a></p>
<p><a href="/css/default.asp">CSS Упражнения</a></p>
```

### Отметки

Линковете могат да бъдат използвани като отметки, за да прескачаме от даден текст до друг в рамките на сайта.

```HTML
<p><a href="#C17">Отиди на Глава 17</a></p>
...
<h2 id="C17">Глава 17</h2>
<p>В тази глава се обясняват някои неща</p>
```

## Списъци

Списъците ни позволяват да групираме сходни елементи. Списъците биват три вида:

- Ненареден (unordered list).
- Нареден (ordered list).
- Списък с описания (description list).

### Ненареден

За създаване на ненареден списък се използва тагът `<ul>`, като стойностите се задават с `<li>`.

```HTML
<ul>
    <li>Кафе</li>
    <li>Чай</li>
    <li>Сок</li>
</ul>
```

#### Промяна на маркера

Използва се свойството `list-style-type`, като то приема следните стойности:
- disc
- circle
- square
- none

### Нареден

За създаване на нареден списък се използва тагът `<ol>`, като стойностите се задават с `<li>`.

```HTML
<ol>
    <li>Кафе</li>
    <li>Чай</li>
    <li>Сок</li>
</ol>
```

#### Промяна на маркера

Използва се свойството `list-style-type`, като то приема следните стойности:
- decimal
- upper-alpha
- lower-alpha  
- upper-roman
- lower-roman

### Списък с описания

Този списък съдържа термини, като към всеки термин има описание. Списъкът се създава чрез тага `<dl>`, терминът с `<dt>`, а описанието с `<dd>`.

```HTML
<dl>
    <dt>Чай</dt>
    <dd>- гореща напитка</dd>
    <dt>Лимонада</dt>
    <dd>- студена напитка</dd>
</dl>
```
