# Конспект

Дисциплината въвежда в основните езици за уеб дизайн и особеностите на тяхното приложение при изграждане на уеб сайтове.

Представят се езиците: HTML, CSS и JavaScript в съответствие със спецификациите на World Wide Web Consortium (W3C), както и изисквания и похвати в уеб дизайна.

## Теми

- **[Въведение в уеб дизайна](web-design/topics/introduction/intro.mdx)**
- HTML: тагове, елементи, атрибути, маркиране, програми, цветове
- Основни тагове в HTML: текст, списъци, линкове, таблици, метаелементи
- Въведение в CSS - синтаксис, начини, класове и идентификатори
- CSS: стилове за форматиране на текст
- CSS: стилове за позициониране на елементи, кутиен модел
- CSS: наследственост, селектори
- Изображения, семантични елементи, мултимедия
- Формуляри - полета, бутони, менюта
- Въведение в JavaScript - синтаксис, функции, събития
- DOM - дърво, достъп, модификации
- Концепции за уеб дизайн, общи правила и етапи
- Критерии за оценка на сайт и основни грешки
- Responsive и Adaptive дизайн
