# Какво е ASP.NET Core

ASP.NET Core е модерен, бърз и между-платформен уеб framework на .NET, използван за създаване на уеб приложения, REST API‑та и облачни услуги.

---

## Характеристики

- **Между-платформен framework** - Работи на Windows, Linux и macOS.
- **Висока производителност** - Един от най‑бързите уеб frameworks.
- **Модулна архитектура** - Използва middleware компоненти вместо тежки монолитни pipelines.
- **Поддръжка на MVC и Razor Pages** - Позволява изграждане на уеб приложения с модели, изгледи и контролери.
- **REST API разработка** - Идеален за backend услуги, microservices и мобилни приложения.
- **Dependency Injection по подразбиране** - Вграден DI контейнер.
- **Конфигурация и логване** - Система за конфигурации и логове.
- **Интеграция с Entity Framework Core** - ORM за работа с бази данни.
- **Поддръжка на Docker и облак** - Идеален за контейнеризация и deploy в Azure, AWS (Amazon Web Services).

---

## Основни разлики между ASP.NET и ASP.NET Core

- **Между-платформеност** - ASP.NET работи само на Windows; ASP.NET Core работи на Windows, Linux и macOS.
- **Производителност** - ASP.NET Core е значително по‑бърз благодарение на новия Kestrel сървър и оптимизации.
- **Архитектура** - ASP.NET е монолитен; ASP.NET Core е модулен и използва middleware pipeline.
- **Open Source** - ASP.NET Core е напълно open‑source; ASP.NET е частично отворен.
- **Hosting модел** - ASP.NET работи върху IIS; ASP.NET Core може да работи върху Kestrel, IIS, Nginx, Apache, Docker.
- **Dependency Injection** - В ASP.NET Core DI е вградено; в ASP.NET трябва външен контейнер.
- **Конфигурация** - ASP.NET Core използва JSON, environment variables, secrets; ASP.NET използва Web.config.
- **Версии и бъдещо развитие** - ASP.NET е в режим поддръжка; ASP.NET Core е активната, развиваща се платформа.
