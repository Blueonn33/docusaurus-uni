# Уеб слой (Web layer)

Уеб слоят представлява входната точка на приложението и отговаря за обработката на входящите HTTP заявки, валидирането на данните, извикването на бизнес логика от приложния слой и връщането на подходящ отговор към клиента. Той не съдържа бизнес логика, а служи като посредник между потребителя и вътрешните слоеве на системата.

---

## Основни отговорности

- Приемане на HTTP заявки чрез контролери.
- Маршрутизиране на заявките към правилните действия чрез Routing.
- Обвързване на данни (Model Binding) от заявката към входни модели.
- Валидация на входните данни чрез Model Validation.
- Извикване на бизнес логика чрез услуги от Application слоя.
- Връщане на отговор:
    - HTML чрез Razor Views.
    - JSON при API контролери.
- Прилагане на функционалности като Authentication, Authorization и филтри.

---

## Елементи на слоя

- **Контролери** - Основният механизъм за обработка на заявки.
- **Views** - Визуализация на HTML съдържание.
- **ViewModels** - Модели, предназначени за визуализация.
- **DTOs** - Модели за вход/изход при API.
- **Filters** - Логика, която се изпълнява преди или след action метод.
- **Routing** - Определя кой контролер и метод да бъдат извикани.

--- 

## Роля в многослойната архитектура

Уеб слоят е най‑външният слой и комуникира директно с клиента. Той:

- Не съдържа бизнес логика.
- Не работи директно с базата данни.
- Не съдържа домейн модели.

Вместо това той:

- Приема заявката.
- Валидира входните данни.
- Извиква услуга от Application слоя.
- Получава резултат.
- Връща HTML или JSON към клиента.

#### Пример

Имаме следния поток на заявка към приложението:

1. Клиентът изпраща заявка към `/books`.
2. Routing я насочва към `BooksController`.
3. Model Binding преобразува входните данни към `ViewModel`.
4. Controller извиква `IBookService` от Application слоя.
5. Service използва `Repository` от Infrastructure слоя.
6. Controller връща View.

```csharp
public class BooksController : Controller
{
    private readonly IBookService _service;

    public BooksController(IBookService service)
    {
        _service = service;
    }

    public IActionResult Index()
    {
        var books = _service.GetAll();
        return View(books);
    }
}
```
