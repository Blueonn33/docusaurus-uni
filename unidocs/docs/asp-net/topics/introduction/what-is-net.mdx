# Какво е .NET - CLR, SDK

## .NET

.NET е между-платформена и безплатна платформа за разработка, създадена от Microsoft, която позволява изграждане на уеб, десктоп, мобилни, облачни, IoT и игрови приложения. Тя включва runtime, библиотеки, компилатори и инструменти, които улесняват разработката и осигуряват висока производителност.

- Работи на Windows, Linux и macOS.
- Поддържа езици като C#, F# и VB.
- Има високопроизводителен runtime с автоматично управление на паметта.
- Предлага богата екосистема от NuGet пакети.
- Подходяща е за уеб (ASP.NET Core), мобилни (MAUI), десктоп (WPF/WinUI), облак (Azure) и др.

---

## CLR

**CLR (Common Language Runtime)** е основното runtime ядро на .NET, тоест средата, която изпълнява C# кода.

### Какво прави CLR

- **Управлява паметта** - Автоматичен Garbage Collector освобождава ненужната памет.
- **Изпълнява IL кода** - Компилира Intermediate Language (IL) до машинен код чрез JIT.
- **Осигурява сигурност** - Контролира достъпа, sandboxing, проверки.
- **Управлява нишки** — threading, task scheduling.

--- 

## SDK

**SDK (Software Development Kit)** е пакет от инструменти, библиотеки и компилатори, които позволяват разработване на приложения за дадена платформа.

### Какво включва едно SDK

- **Компилатор** - Превежда кода в изпълним формат.
- **Runtime** - Средата, която изпълнява кода.
- **Библиотеки** - Готови функционалности, които директно се използват в проекта.
- **Инструменти за build** - MSBuild, CLI команди и др.
- **Документация** - Описания как се използват API‑тата.
