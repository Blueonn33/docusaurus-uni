# Инфраструктурен слой (Infrastructure layer)

Инфраструктурният слой съдържа техническите имплементации, чрез които приложението комуникира с външни ресурси: база данни, файлове, мрежови услуги, имейл системи, кеш, логване и др.
Той реализира абстракциите, дефинирани в Приложния слой, и предоставя конкретни механизми за достъп до данни и външни системи.
Инфрастуктурният слой не съдържа бизнес логика. Неговата роля е да изпълнява технически задачи, които Приложният слой изисква чрез интерфейси.

---

## Основни отговорности

- Имплементира репозитории за работа с база данни.
- Съдържа Entity Framework (EF) Core DbContext и конфигурации.
- Управлява миграции и достъп до данни.
- Имплементира външни услуги (имейл, файлове, API интеграции).
- Осигурява кеширане, логване и други технически механизми.
- Следва интерфейсите от Приложния слой, без да дефинира нови бизнес абстракции.

---

## Елементи на слоя

- **Repositories** - Имплементации на интерфейси за достъп до данни.
- **DbContext** - Основният клас за работа с EF Core.
- **Entity Configurations** - Fluent API конфигурации.
- **Migrations** - Управление на схемата на базата данни.
- **Външни услуги:**
    - Изпращане на имейл.
    - Съхранение на файлове.
    - Външни API интеграции.

---

## Роля в многослойната архитектура

Инфраструктурният слой е най‑вътрешният технически слой, който комуникира с реалните ресурси. Той:
- Имплементира интерфейси, дефинирани в Приложния слой.
- Не се извиква директно от Уеб слоя.
- Не съдържа бизнес логика.
- Не дефинира домейн модели.

**Потокът е следният:**
- Уеб слоят извиква Приложния слой.
- Приложният слой извиква интерфейс (например IBookRepository).
- Инфраструктурният слой предоставя конкретната имплементация (BookRepository).
- Имплементацията използва EF Core, SQL или външни услуги.

Така се постига слабо свързване и лесна подмяна на имплементации.

---

#### Пример

```csharp
public class BookRepository : IBookRepository
{
    private readonly AppDbContext _context;

    public BookRepository(AppDbContext context)
    {
        _context = context;
    }

    public IEnumerable<Book> GetAll()
    {
        return _context.Books.ToList();
    }
}
```

- IBookRepository е дефиниран в Application слоя
- BookRepository е имплементацията в Infrastructure слоя
- AppDbContext е EF Core контекстът
